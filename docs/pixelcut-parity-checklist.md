# Pixelcut Parity Checklist for ListForge

Use this as the execution checklist to make switching from Pixelcut feel frictionless.

## P0 (must-have for switch confidence)

- [ ] Magic Eraser (object removal with brush mask)
- [ ] AI background generation (prompt + quick presets)
- [ ] Full edit sliders in-app:
  - [ ] exposure
  - [ ] contrast
  - [ ] highlights/shadows
  - [ ] temperature/tint
  - [ ] saturation/vibrance
  - [ ] sharpen/denoise
- [x] Upscale 2x/4x
- [x] One-call enhance + upscale endpoint
- [x] Mode-based background options (auto/electronics/general)
- [ ] Before/after slider (not only side-by-side)
- [x] "What changed" badges in results

Exit criteria for P0:
- A Pixelcut user can complete a full listing photo set without leaving ListForge.

## P1 (quality + speed)

- [ ] Apply-to-all consistency mode with one tap
- [ ] Per-photo processing state and retry queue
- [ ] Strong relight presets (studio/daylight/premium gloss)
- [ ] Provider fallback transparency in UI
- [ ] Export presets by marketplace (Facebook, eBay, Craigslist)

Exit criteria for P1:
- User can process 10+ photos consistently in under 3 minutes.

## P2 (advanced differentiation)

- [ ] AI quality score delta (before/after)
- [ ] Auto hero-photo recommendation
- [ ] Listing-readiness score from image quality + required-shot coverage
- [ ] Style memory by user mode/category

Exit criteria for P2:
- ListForge provides both top-tier edits and listing conversion intelligence.
